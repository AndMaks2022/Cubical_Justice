# Cubical Justice / Кубическая справедливость

**Спасите национальное достояние — пельмени.**

Вы — агент №6, сотрудник организации ПУНБ (Поиск утерянных национальных блюд). Ваша задача — добыть и спасти уникальный образец пельменей, который находится под угрозой вымирания. Операция идёт не по плану. Вы в ловушке. Связь с центром потеряна.

Вам предстоит не просто выбраться, но и выполнить задание любой ценой.

## 🎮 Особенности

- Короткая сюжетная головоломка (15–30 минут)
- 5 уровней + интерактивная предыстория
- 8 кат-сцен, в двух из которых можно повлиять на сюжет
- Стильная рисованная 2D-графика
- Система автосохранения (можно отключить в настройках)
- 4 слота сохранения с информацией о дате, времени и уровне
- После прохождения открываются все уровни для повторного прохождения

---

## 🕹️ Управление

- **Движение** — кнопки на экране (или телефонное управление в настройках)
- **Взаимодействие** — кнопка действия
